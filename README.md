# 🗣️ Namma Voice

> **Your Voice. Directly to Action.**

Namma Voice is an AI-powered civic accountability platform designed to bridge the gap between citizens and government authorities. Citizens can report civic issues, track their status, and verify resolutions, while government departments can efficiently manage and resolve complaints.

## 🎯 Problem Statement

Citizens frequently face issues such as:

- Overflowing garbage bins
- Potholes and damaged roads
- Water leakage and drainage issues
- Streetlight failures
- Public safety concerns
- Illegal dumping and encroachments

Existing complaint systems often suffer from:

- Lack of transparency
- Poor accountability
- No public visibility
- No citizen verification mechanism
- Delayed resolutions

Namma Voice aims to solve these challenges through transparency, accountability, and citizen-driven verification.

## ✨ Key Features

### 👤 Citizen Portal

- Register and Login
- Raise complaints
- Upload descriptions and supporting evidence
- Track complaint status
- Verify complaint resolution
- View complaint history

### 🏢 Officer Portal

- Department-specific login
- View assigned complaints
- Update complaint status
- Mark issues as resolved
- Add remarks and updates

### 🛡️ Admin Dashboard

- Monitor all complaints
- Manage departments
- Analyze complaint statistics
- View system-wide performance metrics
- Track resolution efficiency

### 📊 Transparency Dashboard

- Public complaint visibility
- Complaint tracking
- Department performance monitoring
- Real-time civic issue insights

### 🏆 Accountability Leaderboard

Ranks departments based on:

- Resolution Rate
- Average Resolution Time
- Pending Complaints
- Overall Performance

This creates transparency and encourages faster issue resolution.
